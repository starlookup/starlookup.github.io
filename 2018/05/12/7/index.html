Heap memory

What is Heap(什么是堆)

堆是由操作系统分配给每个程序的一块内存区域。不同于栈，堆内存空间是可以动态分配得。而这也就意味着程序可以任意在它需求时从堆段内存，“请求”或者“释放”内存。而且，这块内存是全局共享的。也就是说，它可以在程序的任何地方有效访问和修改，而不是仅仅是在申请调用它的函数里面。程序一般使用指针来实现内存的动态申请，虽然这样效率会比使用本地变量（栈上）稍有下降。

Using dynamic memory(使用动态内存)

stdlib.h 提供了很多可以访问、修改以及管理的标准库函数。常用的函数包括malloc和free:

文档中对于malloc和free的说明：

malloc
free

需要注意的是这些内存分配函数由标准库提供。这些函数在开发人员与操作系统之间提供了一个有效管理堆内存的中间层。开发人员有必要在使用完分配的内存后立即将其free。从内部来看，这些函数都使用了两种系统调用sbrk和mmap来申请或释放操作系统的堆内存。
